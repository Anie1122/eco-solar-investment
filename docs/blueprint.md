# **App Name**: EcoSolar Invest

## Core Features:

- User Registration & Profile: Secure user registration with profile details, including country selection, auto-detected currency, and phone number with country code.
- Wallet Management: Enable deposits via Flutterwave within specified limits, and withdrawals with balance checks. Display insufficient funds alerts.
- Investment Plans Display: Display available solar investment plans with clear details on investment amount, duration, daily profit, and total returns.
- Automated Investment Logic: Deduct investment amounts, credit daily profits automatically using a Cloud Function, unlock bonus after first investment, and mark investments as active/completed.
- Transaction History Tracking: Track all transactions including deposits, withdrawals, bonuses, and profits with real-time updates.
- Notifications: Provide in-app notifications for deposits, investments, profits, withdrawals, and insufficient funds.
- Personalized Investment Plan Suggestions: Use an AI tool to suggest optimal investment plans based on user profile and wallet balance.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), echoing trust and environmental themes.
- Background color: Very light blue (#E6F5FF), a desaturated version of the primary to ensure clarity and readability.
- Accent color: Viridian (#40826D) offering an analogous complement related to solar's organic connections and natural energy.
- Body and headline font: 'Inter', a sans-serif typeface that balances modern design with a neutral and objective appearance. Suited for all text, body and headline.
- Use clean, minimalist icons related to solar energy, investments, and financial transactions.
- Implement a mobile-first, fully responsive layout with a clean, professional fintech design.
- Incorporate subtle animations for transactions and updates to enhance user engagement.
